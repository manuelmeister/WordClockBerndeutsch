## Install

Install with Platformio: ````platformio lib install````

else it needs

- ESP8266WiFi v2.4.2  
- WiFiManager v0.14.0  
- TimeLib v1.5  
- Timezone v1.2.2  
- Adafruit_NeoPixel v1.1.7
